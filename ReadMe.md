# RW-SRS
RW-SRS是一个Rusted Warfare游戏的服务端
旨在用低资源消耗运行可用的游戏服务器

## 运行配置
本项目暂时仅支持linux操作系统
    cpu     内存    硬盘    gcc版本
    任何     2mb    10mb   gcc11及以上
